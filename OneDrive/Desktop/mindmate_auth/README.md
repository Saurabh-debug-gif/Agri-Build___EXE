# 🧠 MindMate – AI Mental Wellness Companion

## 📖 Description
MindMate is a web-based mental wellness application designed to help users
track their mood, engage in empathetic AI-driven conversations, and improve
their emotional well-being through simple daily interactions.

The project was built for a **Google Developer Group (GDG)** event and
integrates **Google Generative AI** concepts for intelligent responses.

---

## ✨ Features
- User authentication (Signup & Login)
- AI-powered mental health chat assistant
- Emotion detection (Happy, Sad, Angry, Neutral)
- Mood tracking and visualization
- Secure session-based authentication
- Offline AI fallback for demo reliability
